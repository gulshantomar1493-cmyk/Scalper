MarketScalper — Madad / Guide

🎯 Ye tool kya hai?

MarketScalper ek market ANALYSIS tool hai — ye khud trade NAHI karta. Ye tumhe batata hai ki kab, kahan aur kyun ek trade ka setup ban raha hai. Order tum khud lagate ho. (Isliye "no execution".)

🧭 Pages (sidebar)

  • Live — real-time chart + analysis (1m/5m).
  • Replay — purane data ko chala ke practice.
  • Trade Review — recommendations ne kaisa perform kiya (display-only).
  • Journal — apne trades/notes/lessons log karo.
  • Analytics — win-rate, PnL, expectancy.
  • Settings — theme, token, symbols.

⏱️ Timeframes

Upar [1m][5m]…[1M] buttons se timeframe badlo. Sirf 1m aur 5m pe analysis (recommendation/plan) dikhta hai — bade timeframes sirf market context ke liye hain (fake analysis kabhi nahi).

🎯 Right panel

  • Recommendation: LONG/SHORT + grade (A+) + confidence % + stars.
  • Trade Plan: Entry / Stop / Target 1&2 / Risk:Reward — sirf SUGGESTION.
  • Market Context: Trend + 4 engines (Structure/Liquidity/Volume/Momentum).

🎨 Theme & 🔰 Beginner mode

Top-right theme button (dark/light). Sidebar me Beginner Mode toggle — helper text on/off.

⚠️ Zaroori

  • Tool khud trade nahi karta.
  • Koi strategy tab tak TRUSTED nahi jab tak 200+ recommendations pe fees ke baad positive na aaye. Signals blindly follow mat karo.

Dobara kholne ke liye ❓ ya sidebar "Need help?". (Band: Esc ya bahar click)
