MarketScalper — Madad / Guide

🎯 Ye tool kya hai?

MarketScalper ek market ANALYSIS tool hai — ye khud trade NAHI karta. Ye tumhe batata hai ki kab, kahan aur kyun ek trade ka setup ban raha hai. Final decision aur order tum khud apne exchange par lagate ho. (Isliye header me likha hai: "no execution".)

📈 Chart

  • Bada chart (upar): 1-minute candles — main price action.
  • Chhota chart (5m context): 5-minute candles — bada trend samajhne ke liye.
  • Chart pe jo lines, boxes, arrows dikhte hain wo engine ke detect kiye structures hain (niche "Overlays" dekho).

▶️ Replay kya hai aur kyun?

Kahan hai: upar header me "▾ Replay & Tools" button dabao — replay controls neeche khul jayenge (home saaf rakhne ke liye ye chhupe rehte hain).

Replay = purane data ko dobara chala ke dekhna, taaki tool ka analysis samajh sako bina live market ka wait kiye. Live me har candle 1 minute me banti hai — replay me poora din seconds/minutes me dekh sakte ho.

  • From / To: kaunsi date-time se kaunsi tak chalana hai.
  • Speed: ×60 = dekhne ke liye best (1 candle/second). ×max sirf fast processing ke liye — itna tez ki screen pe kuch dikhta nahi.
  • Start / Stop: replay shuru / band.

💡 Dense data: April 2026 ki dates use karo (jaise 20-04-2026). Ek session dekhna ho to From 08:00 To 13:00 (LONDON) — ~5 min me pura ×60 par.

🟢 Live vs Replay

LIVE (header me green dot) = abhi ka real Binance data. Replay = history. Replay chalu hone par chart history dikhata hai; khatam hone par wapas live pe aa jata hai.

🎯 Signal kaise banta hai? (Right panel)

Right side ka panel batata hai ki abhi ka setup kitna strong hai:

  • Score /100 + Verdict: setup ki strength.
    • NO_SIGNAL → koi setup nahi / koi gate fail.
    • BELOW_THRESHOLD → weak setup.
    • TRADEABLE (≥75) → acha setup.
    • A+ (≥85) → best setup.
  • Hard Gates (G1–G6): 6 zaroori shartein. Koi ek bhi ✗ (fail) → seedha NO SIGNAL (score dikhta hi nahi). Jaise abhi shayad G3 = session LATE ✗ ho (raat 21:00–00:00 UTC low-liquidity time me tool jaan-boojhkar signal nahi deta). prov = abhi placeholder gate.
  • Signal Agreement: 4 engines milke score banate hain — Structure 30%, Liquidity 30%, Volume 25%, Momentum 15%.
  • Trade Plan: jab TRADEABLE ho tabhi dikhta hai — Entry, Stop-loss, TP1, TP2, suggested quantity, Risk:Reward. Ye sirf SUGGESTION hai; execute tum karte ho.
  • Reason Trace: kyun signal bana ya nahi bana — rule by rule (✓ / ✗).

🖍️ Chart Overlays (kya-kya dikhta hai)

  • Pivots: H / L points (swing high / swing low).
  • Trendlines: green = support, red = resistance.
  • Zones (boxes): Order Blocks (OB) aur Fair Value Gaps (FVG) — wo areas jaha price react kar sakta hai.
  • Arrows: liquidity sweeps (jaha price ne high/low ko "shikaar" kiya).
  • Header me trend dikhta hai: BULLISH / BEARISH / RANGE.

🔍 Audit (Random line / Sweep / OB) — Advanced

Kahan hai: "Replay & Tools" drawer me neeche "⚙ Advanced" par click karo (starter ke liye chhupa rakha hai). Ye learning / verification tool hai — button dabao → ek random detected structure pe chart jump karega → dekho engine ne sahi pakda ya nahi → Accept / Reject. Trade se koi lena-dena nahi — sirf tumhare review ke liye.

🎨 Theme (Light / Dark)

Header me 🌙 Dark / ☀️ Light button se theme badlo. Default light cream hai (aankhon ko aaram); dark chahiye to ek click. Tumhari choice yaad rahegi.

📊 Dashboard & 🔄 Symbol

Dashboard button: Analytics (win rate, expectancy) + Journal (logged trades + reasons). Upar BTCUSDT / ETHUSDT se coin badlo.

⚠️ Zaroori baat

  • Ye tool khud trade nahi karta — sirf analysis + suggestion.
  • Koi bhi strategy tab tak "TRUSTED" nahi jab tak 200+ recommendations par fees ke baad positive result na aaye. Abhi validation phase hai — signals blindly follow mat karo.
